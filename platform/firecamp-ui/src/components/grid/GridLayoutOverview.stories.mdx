## **Grid Layout**

To create the layout for page in proper structure for easy maintenance.

#### **Component Usage**:

We are using flex based Row [FlexLayout] - Columns [ContainerLayout] layout. <br/>

| (Row) Classname | Usage                                                                       |
| --------------- | --------------------------------------------------------------------------- |
| flex            | [true] the row expand to occupy all the available space within its parent.  |
|                 | [false - default] to adjust its own size based on the size of its contents. |

| (Column) Prop | Usage                                                                                     |
| ------------- | ----------------------------------------------------------------------------------------- |
| flex          | [true] to occupy all the available space within its parent (when used within FlexLayout). |
|               | [false - default] to adjust its own size based on its contents.                           |

#### **Component Related**:

> Layout is created with : GridLayout, FlexLayout, ContainerLayout.

#### **Component Dependencies**:

- Use tailwind classes for flex to generate any layout you want.

#### **Best Practices**:

- Use `<GridLayout></GridLayout>` as wrapper
- Always nest `<ContainerLayout></ContainerLayout>` in `<FlexLayout></FlexLayout>`

Do:

```
<GridLayout>
	<FlexLayout>
		<ContainerLayout>
		Loream ipsum unde omnis iste natus error sit voluptatem accusantium.
		</ContainerLayout>
		<ContainerLayout>
			<FlexLayout>
				<ContainerLayout>
					Loream ipsum unde omnis iste natus error sit.
				</ContainerLayout>
      			<ContainerLayout>
					Loream ipsum unde omnis iste natus error sit.
				</ContainerLayout>
			</FlexLayout>
		</ContainerLayout>
	</FlexLayout>
</GridLayout>
```

Avoid:

```
<FlexLayout>
  <FlexLayout>
    Do not nest unnecessary Rows
  </FlexLayout>
</FlexLayout>
```

```
<FlexLayout>
	<ContainerLayout>
		<ContainerLayout>
			Do not nest unnecessary Rows
		</ContainerLayout>
	<ContainerLayout>
</FlexLayout>
```

```
<FlexLayout>
	Do not place any component directly in Row, use Column inside a Row.
</FlexLayout>
```
